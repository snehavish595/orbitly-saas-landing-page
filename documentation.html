Orbitly Documentation

Welcome! This guide covers everything you need to install, customize, and launch the Orbitly template. No build tools or Node.js required — it's pure HTML, CSS, and JavaScript.

✅ Thank you for purchasing Orbitly. If you find it helpful, please leave a rating on Envato — it means the world to indie creators!

Overview

Orbitly is a premium single-page landing page template designed for SaaS products, project management tools, and creative agency software. It features:

  • Dark-mode-first design with amber/emerald accent palette
  • Fully responsive — mobile, tablet, and desktop optimized
  • Pure HTML/CSS/JS — no build step, no dependencies to install
  • CSS custom properties (design tokens) for one-line rebranding
  • Scroll-triggered animations using IntersectionObserver
  • Interactive pricing toggle (monthly/annual) with live price swap
  • FAQ accordion with accessible ARIA attributes
  • Animated counter statistics
  • Infinite logo marquee
  • Sticky glassmorphism navbar
  • Mobile hamburger menu with smooth slide-in panel
  • Working email validation on the CTA form
  • Scroll-to-top button

Quick Start

Option A — Open directly

Just open index.html in any modern browser. No server needed.

Option B — Local dev server (recommended)

For a smoother development experience, serve it with any static server:

# Using Node (npx)
npx serve .

# Using Python 3
python -m http.server 8080

# Using VS Code
Install the "Live Server" extension, then click "Go Live"

Colors & Design Tokens

All design decisions live in CSS custom properties at the top of style.css. To rebrand the template, edit these variables:

:root {
  /* ── Core surfaces ───────────── */
  --ink:        #0d0f14;   /* Page background */
  --ink-light:  #1a1e2a;   /* Secondary bg */
  --surface:    #12151f;   /* Sidebar/panels */
  --card:       #181c28;   /* Card background */
  --card-hover: #1f2436;   /* Card hover bg */
  --border:     rgba(255,255,255,.07);
  --border-hi:  rgba(255,255,255,.14);

  /* ── Brand accent colors ──────── */
  --amber:      #f59e0b;   /* PRIMARY accent */
  --amber-dark: #d97706;   /* Button hover */
  --emerald:    #10b981;   /* Success/positive */
  --indigo:     #6366f1;   /* Secondary accent */
  --sky:        #38bdf8;   /* Tertiary accent */
}
💡 Quick rebrand tip: Change --amber to your brand color (e.g., #3b82f6 for blue, #ec4899 for pink) and the entire template updates instantly.

Typography

Orbitly uses two Google Fonts loaded in index.html:

  • Cabinet Grotesk (900, 800, 700) — Headings, logo, numbers
  • Satoshi (400, 500, 700) — Body text, UI labels

To swap fonts, update the <link> tag in the <head> and the font-family declarations in style.css.

Sections Guide

Changing the logo

Search for Orbitly in index.html to find all logo instances (navbar and footer). Update the text and optionally replace the Font Awesome icon inside .logo-icon.

Updating navigation links

Edit the <ul class="nav-list"> inside the <header>. All href values use anchor IDs matching section IDs (e.g., #features scrolls to <section id="features">).

Trusted brands marquee

Find the .marquee-track element and update the <span> text labels. The track is duplicated to create a seamless loop — always keep the full list duplicated.

Stats numbers

Each counter has a data-count attribute that drives the animation:

<div class="stat-number" data-count="12000">0</div>

Feature cards

Add or remove .feat-card elements inside .feature-cards-grid. Icons use Font Awesome classes. The grid auto-adjusts to 2 columns.

Testimonial cards

Each .t-card can be duplicated. Update the gradient in style on .t-avatar for unique author colors.

Blog thumbnails

Blog cards use CSS gradient backgrounds (.art-1 through .art-4). To use real images, replace the <div class="blog-thumb-art art-N"> with an <img> tag:

<div class="blog-thumb">
  <img src="assets/images/blog1.jpg" alt="Post title" loading="lazy">
  <span class="blog-tag">Category</span>
</div>

Team section

The template doesn't include a team section by default to keep the layout flexible. To add one, copy the pattern from the previous version and paste it before the Blog section. Add team images in assets/images/.

Pricing Toggle

The monthly/annual toggle is driven by data-monthly and data-annual attributes on each .p-amount element:

<span class="p-amount" data-monthly="49" data-annual="34">$49</span>

The JavaScript in main.js swaps these automatically when the toggle is checked. For free-tier cards, set both values to 0 and the script will always display $0.

Animations

Any element with a data-animate attribute will fade in when scrolled into view:

<!-- Fade up (default) -->
<div data-animate="fade-up">...</div>

<!-- Slide in from right -->
<div data-animate="fade-left">...</div>

<!-- Stagger delay (CSS variable) -->
<div data-animate="fade-up" style="--delay: .2s">...</div>

The observer uses a 12% threshold with a 40px bottom margin. To disable animations for a specific element, simply remove the data-animate attribute and add opacity: 1; transform: none; in CSS.
